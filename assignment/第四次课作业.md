

# 一、我的推荐研究方向


## 中文名称

序列推荐


## English Name

Sequential Recommendation


## 方向介绍

序列推荐主要研究如何根据用户过去的行为序列，预测用户下一步可能感兴趣的物品。


例如：

用户历史行为：

```
电影A → 电影B → 电影C → ?
```


模型根据历史行为预测用户下一次可能观看的电影。


该方向关注：

- 用户兴趣变化
- 行为顺序关系
- 长期兴趣和短期兴趣建模

# 三、相关方法


## 方法1：Transformer推荐方法


代表模型：

SASRec


基本思想：

利用Transformer中的Self-Attention机制学习用户行为之间的关系。


优势：

- 可以捕捉长期兴趣
- 能处理用户行为序列


---

## 方法2：循环神经网络推荐方法


代表模型：

GRU4Rec


基本思想：

使用循环神经网络学习用户行为顺序。


优势：

- 能建模时间顺序
- 适合序列数据分析

# 四、任务与数据形式


## 推荐任务


下一物品预测：

Next Item Prediction


任务目标：

根据用户历史交互记录，预测用户未来可能交互的物品。


---

## 数据形式


用户-物品交互序列：

例如：


```
User1:

item1 → item5 → item8 → ?
```


常见数据：

- 用户ID
- 物品ID
- 交互时间
- 行为类型

# 五、评价指标


## 1. HR@10


Hit Ratio@10


含义：

判断真实目标物品是否出现在推荐列表前10。


数值越高，表示推荐命中率越高。



---

## 2. NDCG@10


Normalized Discounted Cumulative Gain


含义：

评价推荐列表排序质量。


不仅考虑是否推荐正确，还考虑正确物品的位置。
