# 知识库

本仓库用于记录我的学习与研究训练过程。

主要记录假期学习计划、每日学习日志、课程作业、知识库整理以及论文阅读过程。


## 仓库目标

- 能够对简单的论文实验进行复现
## 项目结构

- `assignment/`：课程作业与阶段任务记录。
  
- `daily_logs/`：每日学习日志，用于记录学习内容、问题、总结和下一步计划。

- `knowledge_base/`：长期知识库，按照主题整理推荐系统学习资料。

- `learning_plan.md`：假期学习计划，记录学习目标、阶段安排和任务规划。

- `README.md`：项目说明文件，用于介绍仓库目标、结构和使用方式。


### knowledge_base 文件说明

- `python.md`：Python 语法、数据处理、实验工程实践。

- `math.md`：数学基础、概率统计、优化方法。

- `recsys.md`：推荐系统概念、算法原理、评价指标与工程框架。

- `paper_reading.md`：推荐系统论文阅读方法、论文笔记模板与论文总结。

- `tools.md`：开发工具、实验环境、代码管理和 Obsidian 使用技巧。
