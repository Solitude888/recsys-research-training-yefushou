# 二、Conda环境创建

## 学习内容

学习使用 Conda 管理不同实验环境。

创建 SASRec 专用环境：


conda create -n sasrec python=3.10


激活环境：

conda activate sasrec

# 五、遇到的问题及解决


## 问题

运行PyTorch时出现：

```
WinError 127
c10_cuda.dll加载失败
```


## 原因

之前使用 Anaconda base 环境中的 CUDA 版本 PyTorch。

当前电脑环境无法匹配 CUDA。


## 解决方法

- 创建新的 sasrec 环境
- 安装 CPU 版本 PyTorch
- 使用 sasrec 环境运行实验
